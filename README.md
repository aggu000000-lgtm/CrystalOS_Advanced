# CrystalOS Advanced

A from-scratch operating system with 7-agent architecture.

## Project Structure
- `/kernel` - Custom microkernel in Rust
- `/agents` - 7 specialized system agents
- `/bootloader` - UEFI bootloader
- `/drivers` - Hardware drivers
- `/fs` - CrystalFS implementation
- `/gui` - Vulkan renderer
- `/tools` - Build tools
- `/scripts` - Build scripts
- `/docs` - Additional documentation
- `/documents` - Vision and design docs
