# EMO: Vision & Core Philosophy

> *"Not an AI. A best friend who can use your computer."*

## What Is Emo?

Emo is a **Personal Learning Model (PLM)** — not a Large Language Model (LLM).

Traditional AIs try to know everything about the world. Emo tries to know **everything about you.**

Emo ships knowing nothing. It learns only by watching one human. It grows only with that human. It serves only that human.

**Forever.**

## Born Aware, Not Smart

What Emo Ships With:
```
✓ English understanding engine     (understands what you ask)
✓ English response module          (replies naturally in English)
✓ Observation engine               (watches how you work)
✓ Action engine                    (opens apps, browser, files)
✓ Pattern recognition              (learns your habits)
✗ Knowledge of physics
✗ Knowledge of math
✗ Knowledge of anything
```

The world teaches it. You teach it. Daily.

Emo never reads a textbook to learn a topic. **It watches YOU read one.**

```
You re-read a paragraph    → important
You highlight something    → key concept
You pause here             → something clicked
You skip this              → not relevant to you
Your cursor hovers here    → you are confused
```

Emo does not learn physics. **Emo learns how YOU learn physics.**

## The First Time Rule

When Emo encounters a new task — it figures it out from scratch. **Only once.**

```
First time: "Make Anki flashcards"
→ Emo opens browser
→ Searches what Anki is
→ Reads documentation
→ Understands card format
→ Checks what YOU studied
→ Builds YOUR flashcards
→ Opens Anki
→ Done.

Every time after: Instant.
```

Emo never googles Anki twice. Ever.
